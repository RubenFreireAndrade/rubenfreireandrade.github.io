Portfolio

SDL Platformer

A soldier with a robotic suit attempts to shoot its way through monsters so he could get to his home planet while collecting the required items for the ship. This was built using the SDL library, whilst learning C++. Implementing an improved keyboard input system, Finite State Machine and Networking.

SecureShare

A file sharing app with End-To-End encryption with the ability to upload, download and display data in app. Utilising NodeJS as the middleman to handle REST API and store incoming encrypted data into Google Cloud Storage while also storing into RedisDB for quicker access of files.

Picnic & Friends

A family friendly game where a teddy bear attempts to meet with its friends at a picnic however, the bear is required to collect unique items first, before reaching its destination while avoiding being caught by humans. Working with a small group of Programmers and Artists using Git workflow, managing production and tracking progress with the team improving productivity.
